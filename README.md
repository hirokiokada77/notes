# Markdown Notes

![Screenshot of Markdown Notes](screenshot.png)

**Markdown Notes** is a web app that displays previews of notes written
in Markdown and converts them into [URI
fragments](https://en.wikipedia.org/wiki/URI_fragment), making it easy
to save them quickly by bookmarking or share them with others. You can
also save your Markdown notes directly to your browser.

To run it locally:

```sh
bun install
bun run build
bun run preview
```
